반복문


for
-반복의 기준이 되는 변수(주로 인덱스)를 활용하여 특정 구문부를 반복 실행하고자 할때 사용
	for ([변수 선언 및 초기화]; [반복 조건]; [변수 가감 및 할당]) {
		[구현부]
	}
	   ⅰ. for 반복문을 위한 변수 선언 및 초기화를 실행
	   ⅱ. 반복 조건이 true인지 확인
		*반복 조건이 true라면 구현부를 1회 실행 후 ⅲ.으로 이동
		*false라면 for반복문을 즉시 종료
	   ⅲ. 변수 가감 및 할당을 실시하고 ⅱ.로 돌아간다.

향상된 for (Enhanced for)
-반복할 수 있는 대상이 가지는 인자에 대해 반복하기 위해 사용
-인덱스가 필요 없을 경우 인덱스 범위 오류 최소화 및 가독성 향상 등을 위하여 가능한한 향상된 for반복문을 사용하는 것이 권장
	for ([변수 선언] : [반복 대상]) {


while
-주어진 조건이 true일때 횟수와 관계 없이 구현부를 실행하기 위해 사용
	while ([조건]) {
		[구현부]
	}
	   *만약 조건이 항상 true거나, 거짓으로 변하는 경우의 수가 없다면, 혹은 break 및 return 등의 중단 키워드가 없는 경우 무한히 반복됨으로 유의

do while
-while 반복문과 유사하나 조건과 관계 없이 최초 한번은 반드시 구현부를 실행한다는 차이점이 있다.
	do {
		[구현부]
	} while ([조건]);
	   *만약 조건이 항상 true거나, 거짓으로 변하는 경우의 수가 없다면, 혹은 break 및 return 등의 중단 키워드가 없는 경우 무한히 반복됨으로 유의
