cursor
-마우스 포인터 모양을 지정하기 위한 속성
-형태
	*auto : 기본 값으로, 브라우저가 상황에 맞게 알아서 판단
	*none : 마우스 포인터 없음
	*default : 기본 마우스 상태
	**pointer : 손가락모양
	*progress : 기본 마우스+모래시계
	*wait : 모래시계
	*text : 
	*not-allowed : 금지표시
	*zoom-in : +
	*zoom-out : -
	*url(x) 함수 : x는 원하는 이미지의 주소나 경로.

--------------------------------------------------------------

pointer-events
-마우스 포인터와 요소간의 상호 작용에 관한 설정을 위한 속성이다.
-형태
	*auto : 기본값으로, 상호작용 가능
	*none : 상호작용 불가능

--------------------------------------------------------------

user-select
-사용자의 텍스트 선택에 대한 설정을 위한 속성이다.
-형태
	*auto : 기본값으로, 브라우저가 상황에 따라 알아서 판단
	*all : 요소의 내용이 한번에 선택된다.
	*none : 요소의 내용을 선택 할 수 없음
	*text : 요소의 내용을 일반적으로 선택할 수 있음

--------------------------------------------------------------

z-index
-z축으로의 배치 순서를 설정하기 위한 속성. 값이 클수록 사용자에게 가까이 배치(화면 위로 올라온다.)
-단, position 속성 값이 static인 요소는 해당 속성을 무시한다.
-형태
	*auto : 기본 값으로, 맥락(context)을 형성하지 않는다.
	*[정수]
