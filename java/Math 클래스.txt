Math 클래스
-java.lang.Math 클래스는 수학과 관련된 편의 기능을 제공하는 유틸리티 클래스

-----------------------------------------------------

정적 멤버 변수
-E : 자연 상수(e)
-PI : 원주율

-----------------------------------------------------

정적 메서드
-abs(x) : 숫자 x의 절댓값을 반환
-absExact(x) : 정수 x의 절댓값을 반환. 단, 오버플로우가 발생할 경우 ArithmeticException 예외가 발생
-addExact(a, b) : 정수 a와 b의 합을 반환. 단, 오버/언더플로우가 발생할 경우 ArithmeticException 예외가 발생
-subtractExact(a, b) : 정수 a와 b의 차를 반환. 단, 오버/언더플로우가 발생할 경우 ArithmeticException 예외가 발생
-multiplyExact(a, b) : 정수 a와 b의 곱을 반환. 단, 오버/언더플로우가 발생할 경우 ArithmeticException 예외가 발생
-multiplyFull(a, b) : 일반 정수 (int) a와 b의 곱을 큰 정수(long)로 반환
-negateExact(x) : 정수 x의 부호를 반전하여 반환. 단, 오버/언더플로우가 발생할 경우 ArithmeticException 예외가 발생
-decrementExact(x) : 정수 x에서 1을 뺀 값을 반환. 단, 언더플로우가 발생할 경우 ArithmeticException 예외가 발생
-incrementExact(x) : 정수 x에서 1을 더한 값을 반환. 단, 오버플로우가 발생할 경우 ArithmeticException 예외가 발생
-ceil(x) : double인 x의 소수부를 올림하여 반환
-floor(x) : double인 x의 소수부를 내림하여 반환
-round(x) : double인 x의 소수부를 반올림하여 반환
-max(a, b) : 숫자 a와 b중 큰 값을 반환
-min(a, b) : 숫자 a와 b중 작은 값을 반환
-pow(a, b) : 숫자 a의 b제곱을 반환
-sqrt(x) : 숫자 x의 제곱근을 반환
-cbrt(x) : 숫자 x의 세제곱근을 반환
-log(x) : 숫자 x의 자연로그를 반환
-log10(x) : 숫자 x의 상용로그를 반환
-rendom( ) : 0이상 1미만의 무작위 실수를 반환
-toIntExact(long x) : 큰 정수(long)인 x를 정수(int)로 변환하여 반환. 단, 오버/언더플로우가 발생할 경우 ArithmeticException 예외가 발생
