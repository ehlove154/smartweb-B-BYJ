flex
-플렉스 컨테이너는 이가 가지는 자식 요소들에 대한 배치를 보다 유연하고 쉽게하기 위해 사용
-플렉스 컨테이너가 될 요소들의 display 속성 값을 flex 혹은 inline-flex로 설정

----------------------------------------------------------------

플렉스 컨테이너 (부모)

align-content
-플렉스 컨테이너의  flex-wrap 속성 값이 wrap 일때만 유효하고, 플렉스 아이템 전체에 대해 flex-direction의 '수직한 방향'으로의 정렬을 위한 속성
-형태
	*flex-start : 흐름의 시작 방향에 맞추어 정렬
	*flex-end : 흐름의 끝 방향에 맞추어 정렬
	*center : 가운데 정렬
	*space-between : 각 플렉스 아이템 사이에 균등한 공간을 배분하도록 정렬
	*space-around : 각 플렉스 아이템 사이에 균등한 공간을 배분하고, 각 끝 외부에 각 플렉스 아이템 사이의 공간의 절반에 해당하는 공간을 배분하도록 정렬
	*space-evenly : 각 플렉스 아이템 사이 및 각 끝 외부에 균등한 공간을 배분하도록 정렬

align-items
-플렉스 컨테이너의 flex-direction에 대해 '수직한 방향'으로의 정렬을 위한 속성
-형태
	*flex-start : 흐름의 시작 방향에 맞추어 정렬
	*flex-end : 흐름의 끝 방향에 맞추어 정렬
	*center : 가운데 정렬
	*space-between : 각 플렉스 아이템 사이에 균등한 공간을 배분하도록 정렬
	*space-around : 각 플렉스 아이템 사이에 균등한 공간을 배분하고, 각 끝 외부에 각 플렉스 아이템 사이의 공간의 절반에 해당하는 공간을 배분하도록 정렬
	*space-evenly : 각 플렉스 아이템 사이 및 각 끝 외부에 균등한 공간을 배분하도록 정렬
	*stretch : flex-direction에 수직한 방향으로의 플렉스 컨테이너의 크기에 맞게 늘이거나 줄이도록 정렬.

flex-direction
-플렉스 컨테이너에 부여하는 속성으로, 이가 포함하고 있는 아이템들의 흐름의 방향을 지정하기 위해 사용
-형태
	*column : 세로 방향.
	*column-reverse : 세로 역방향. 요소의 순서도 반전 됨.
	*row : 가로 방향.
	*row-reverse : 가로 역방향. 요소의 순서도 반전 됨.

justify-content
-플렉스 컨테이너의 flex-direction에 대해 '수평한 방향'으로의 정렬을 위한 속성
-형태
	*flex-start : 흐름의 시작 방향에 맞추어 정렬
	*flex-end : 흐름의 끝 방향에 맞추어 정렬
	*center : 가운데 정렬
	*space-between : 각 플렉스 아이템 사이에 균등한 공간을 배분하도록 정렬
	*space-around : 각 플렉스 아이템 사이에 균등한 공간을 배분하고, 각 끝 외부에 각 플렉스 아이템 사이의 공간의 절반에 해당하는 공간을 배분하도록 정렬
	*space-evenly : 각 플렉스 아이템 사이 및 각 끝 외부에 균등한 공간을 배분하도록 정렬

flex-wrap
-플렉스 컨테이너가 가지고 있는 아이템을 한 줄에 표시할 것인가에 대한 설정을 위한 속성
-형태
	*nowrap : 기본값으로 플렉스 아이템이 컨테이너를 벗어나도 한 줄에 표시한다.
	*wrap : 플렉스 아이템이 컨테이너를 벗어나는 경우 여러줄에 걸쳐 표시한다.

gap
-플렉스 컨테이너가 포함하는 아이템간의 거리를 지정하기 위해 사용
-형태
	*[크기 값] : 행간, 열간 거리를 모두 지정
	*[크기 값v] [크기 값h] : 행간 거리를 v로, 열간 거리를 h로 지정

-----------------------------------------------------------------

플렉스 아이템(자식)

flex-grow
-플렉스 아이템이 플렉스 컨테이너 안에서 flex-direction 방향에 따라 남는 여유 공간을 차지하기 위해 늘어날 비율을 지정. 단, 값이 0이면 늘어나지 않는다.
-형태
	*[0 | 자연수] : 기본 값 0

flex-shrink
-플렉스 아이템의 크기가 플렉스 컨테이너보다 클때 플렉스 아이템의 크기를 줄이는 비율을 지정하기 위해 사용
-형태
	*[0 | 자연수] : 기본 값 1

flex-basis
플렉스 아이템의 (기본)크기를 설정하기 위해 사용하는 속성이다. 부모인 플렉스 컨테이너의 flex-direction 속성 값이 row나 row-reverse 일때 이는 width와 같고, column이나 column-reverse일때 이는 height와 같다. 단, 우선 순위는 낮다.
-형태
	*auto
	*[크기 값]

align-self
-부모인 플렉스 컨테이너의 flex-direction 속성에 대해 스스로의 수직한 방향으로의 정렬을 성정하기 위한 속성
-형태
	*flex-start : 흐름의 시작 방향에 맞추어 정렬
	*flex-end : 흐름의 끝 방향에 맞추어 정렬
	*center : 가운데 정렬
	*space-between : 각 플렉스 아이템 사이에 균등한 공간을 배분하도록 정렬
	*space-around : 각 플렉스 아이템 사이에 균등한 공간을 배분하고, 각 끝 외부에 각 플렉스 아이템 사이의 공간의 절반에 해당하는 공간을 배분하도록 정렬
	*space-evenly : 각 플렉스 아이템 사이 및 각 끝 외부에 균등한 공간을 배분하도록 정렬
	*stretch : flex-direction에 수직한 방향으로의 플렉스 컨테이너의 크기에 맞게 늘이거나 줄이도록 정렬.
