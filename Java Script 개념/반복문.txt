반복문


키워드

for
-반복의 기준이 되는 변수(주로 인덱스)를 활용하여 특정 구현부를 반복하고자할 때 사용한다.
	for ([변수 선언 및 초기화]; [반복 조건]; [변수 가감 혹은 복합 대입]) {
		[구현부]
	}
	   1. for 반복문을 위한 변수를 선언 및 초기화 한다.
	   2. 반복 조건이 true인지 확인한다.
		*반복 조건이 true이라면 구현부를 1회 실행하고 (3번)으로 이동한다.
		*반복 조건이 false이라면 반복문 실행을 종료한다.
	   3. 변수 가감 및 복합 대입을 실시하고 (2번)으로 돌아간다.

for ... in
-문자열로 키가 지정된 열거 가능한 대상의 키를 반복하기 위해 사용
-단 반복 대상이 배열일 경우 인덱스를 반복하지만 이를 목적으로 사용하지는 않는다.
	for ([변수 혹은 상수 선언] in [반복 대상]) {
		[구현부]
	}

for ... of
-반복할수 있는 대상이 가지는 인자의 값에 대해 반복하기 위해 사용한다.
-인덱스에 대한 변수가 필요 없을 경우 인덱스 범위 오류 및 가독성 향상 등을 위해 가능한 일반 for 반복문 보다 for ... of 반복문을 사용하는 것이 좋다
	for ([변수 혹은 상수 선언] of [반복 대상]) {
		[구현부]
	}

while
-주어진 조건이 true일때 횟수와 관계 없이 구현부를 실행하기 위해 사용
	while ([조건]) {	
		[구현부]
	}
	   *만약 조건이 항상 true 이거나, false로 변하는 경우의 수가 없다면, 혹은 break 및 return등의 중단 키워드가 없는 경우 무한히 반복됨으로 유의해야 한다.

do ... while
-while 반복문과 유사하나 조건과 관계없이 최초 한 번은 반드시 구현부를 실행한다는 차이점이 있다.
	do {
		[구현부]
	} while ([조건]);
