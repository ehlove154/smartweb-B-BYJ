@keyframes
-css에서 애니메이션을 작성하기 위해 사용하는 쿼리. from부터 to까지 혹은  0%부터 100%까지 애니메이션의 한 사이클에 대한 내용을 담는다.
-작성예시는 아래와 같다.
        @keyframes [애니메이션이름] {
            form {
                ...
            }
            to {
                ...
            }
        }

-------------------------------------------------------------------------------------

animation-duration
-한 사이클이 완성되는데 걸리는 시간을 지정하기 위한 속성
-형태
	*[시간 값]

-------------------------------------------------------------------------------------

animation-iteration-count
-애니메이션의 반복 횟수를 지정하기 위해 사용
-형태
	*infinite
	*[자연수]

-------------------------------------------------------------------------------------

animation-name
-@keyframes쿼리에서 작성한 애니메이션 이름을 값으로 사용하여 선택자에 의해 선택된 요소에 애니메이션을 적용
-형태
	*name : 기본 값으로, 애니메이션 없음
	*[애니메이션 이름]

-------------------------------------------------------------------------------------

animation-paly-state
-애니메이션 재생 상태를 지정하기 위해 사용
-형태
	*running : 기본 값으로, 재생
	*paused : 일시정지

-------------------------------------------------------------------------------------

animation-timing-function
-애니메이션이 적용되는데 있어, 시간의 흐름에 따른 사이클 완료율을 계산하는 함수를 지정하기 위한 속성
-형태
	**ease : 기본 값
	*ease-in
	*ease-in-out
	*ease-out
	**linear
	*[cubic-bezier 함수]

-------------------------------------------------------------------------------------

animation-delay
-애니메이션 최초 사이클이 시작되기 전 지연시킬 시간을 지정. 사이클 사이의 시간에는 영향을 미치치 않는다.
-형태
	*[시간 값]
