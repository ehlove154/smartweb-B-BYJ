grid
-그리드 컨테이너는 이가 가지는 자식 요소(그리드 아이템)들에 대해 반복되는 격자 배치를 보다 쉽게 하기 위해 사용
-그리드 컨테이너가 될 요소의 display 속성 값을 grid 혹은 inline-grid로 설정

------------------------------------------

그리드 컨테이너(부모)

gird-auto-flow
-그리드 아이템을 배치함에 있어 행을 먼저 완성할지, 열을 먼저 완성할지에 대해 설정하기 위한 속성
-값
	*row : 기본 값으로, 행 먼저 채움
	*column : 열 먼저 챙움
-형태
	*[값]

grid-template-columns
-그리드 컨테이너에 부여하는 속성으로, 그리드 열 구조를 설정하기 위한 속성. 값을 공백으로 구분하여 여러개 작성할 수 있다.
-값
	*크기 값
	*fr 값
-형태
	*[값 ...]

grid-template-rows
-그리드 컨테이너에 부여하는 속성으로, 그리드 행 구조를 설정하기 위한 속성. 값을 공백으로 구분하여 여러개 작성할 수 있다.
-값
	*크기 값
	*fr 값
-형태
	*[값 ...]

gap
-그리드 컨테이너가 포함하는 아이템간의 거리를 지정하기 위해 사용
-형태
	*[크기 값] : 행간, 열간 거리를 모두 지정
	*[크기 값v] [크기 값h] : 행간 거리를 v로, 열간 거리를 h로 지정


------------------------------------------

그리드 아이템(자식)

grid-column
-해당 아이템이 차지해야하는 열의 번호의 범위를 지정하기 위해 사용
-형태
	*[시작 번호] / [끝 번호]

grid-row
-해당 아이템이 차지해야하는 행
의 번호의 범위를 지정하기 위해 사용
-형태
	*[시작 번호] / [끝 번호]

order
-해당 아이템의 태그 순번과 관계 없이, 시각적으로 배치되어야 하는 순서를 지정하기 위해 사용
-단, grid-column, grid-row 속성이 적용되어 있다면, order속성은 무시된다.
-속성 값이 같은 아이템끼리는 태그의 배치 순서에 따라 표시 된다.
-형태
	*[숫자] : 기본 값은 0.
