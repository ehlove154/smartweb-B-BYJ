TAZA - Taxi call service

'TAZA - 택시 호출 서비스'는 Kakao Map API를 사용하여 택시를 호출하고, 목적지까지의 예상 소요 시간, 예상 요금을 확인하고 결제까지 가능하도록 구현한 사이트이다. localStorage를 활용해 호출 내역을 확인할 수 있다. 'TAZA - Taxi Call Service' is a site that allows you to call a taxi using the Kakao Map API, check the estimated time to the destination, the estimated fare, and even make a payment. You can check the history using localStorage.

Travel Info

'Travel Info'는 날씨 API를 사용해 여행을 가고 싶은 지역의 현재 날씨와 향후 5일간의 날씨를 확인 할 수 있게 구현했다. 또한, Kakao Map API를 사용하여 관광명소, 음식점, 숙박에 관련된 추천장소를 확인할 수 있다. 'Travel Info' uses the weather API to check the current weather and the weather for the next 5 days in the area you want to travel to. In addition, you can check recommended places related to tourist attractions, restaurants, and accommodations using the Kakao Map API.
